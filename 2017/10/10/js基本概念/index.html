数据类型

基本数据类型5种：Undefined、Null、Boolean、Number、String。

复杂数据类型1种：Object（无序的名值对组成）

  1. Undefined类型

    只有一个值：undefined。

注意： 未初始化的变量和未定义的变量不一样

注意：但是他们的typeof都是undefined

  1. Null类型

只有一个值：null。
null派生子undefined：

如果定义的变量将来用于保存一个对象，那么最好将该变量初始化为null，可以体现null作为空指针的惯例，也有助于区分null和undefined。

  1. Boolean类型

有两个值：true、false。

true不一定等于1，false不一定等于0。但是别 的类型的值都可以通过转型函数Boolean(),来转换成对应的Boolean值。

数据类型 转换为true的值 转换为false的值
Boolean true false
String 任何非空字符串 ‘’（空字符串）
Number 任何非0数值（包括无穷大） 0、NaN
Object 任何对象 null
Undefined 不适用 undefined

有时候（如if语句）会自动执行Boolean()转换

  1. Number类型

    8进制整数以0开头，，若数值超出范围，就忽略前面的0，当做十进制处理。十六进制以0x开头。

浮点数包含小数点，小数点后面至少有一位数字。

浮点数值的最好进度17位小数，但计算精度不如整数：0.1+0.2=0.30000000000000004。这是因为基于IEEE754数值的浮点计算。所以不要测试两个特定的浮点数值。

范围：最小值保存在Number.MIN_VALUE中，最大值在Number.MAX_VALUE。若计算结果超过这个范围则为Infinity或-Infinity，该值无法再参与计算。

NaN：非数值，任何数值除以0返回NaN，不会影响其他代码执行。任何涉及NaN的操作都返回NaN，NaN与任何值都不相等（包括NaN）。
